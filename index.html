¡Hola! me llamo Javier y soy un Software Developer

Para mi programar es un arte, y amo lo que hago ...

Recuerdo la primera vez que utilicé una computadora cuando era niño; quedé fascinado. Pasé numerosas horas de mi infancia descargando e instalando aplicaciones, y en ese momento supe que debía aprender a construir cosas tan asombrosas como aquellas.

Durante los últimos cuatro años de mi vida, me he dedicado a la creación de aplicaciones web, trabajando tanto en empresas privadas como en algunos proyectos personales por hobby. Esta experiencia ha fortalecido mi habilidad para materializar ideas y crear soluciones innovadoras.


Estos son algunos de los últimos proyectos en los que he trabajado:

Factura.com

Desarrollo del sitio comercial actual de factura.com, esta implementación tiene como objetivo cautivar la atención de potenciales clientes y culminar con la compra de un plan.

Tecnologias usadas: Nuxt (Vue.js) + Tailwind + HTML

API Docs

Desarrollo de la documentación para el API de factura.com asi como la implementación y mejora de multiples endpoints del backend.

Tecnologias usadas: Vuepress (Vue.js) + Laravel

Formación Docente

Proyecto desarrollado para la UDG, esta aplicación consiste en una plataforma que sirve para administrar cursos y que los profesores puedan tomarlos.

Tecnologias usadas: Laravel + Vue.js + Vuesax 2

Messenger RealTime

Proyecto en el cual se permite enviar y se recibir mensajes a diferentes contactos en tiempo real, con notificaciones.

Tecnologias usadas: Laravel + Vue.js + Bootstrap vue + Pusher

Targuage.com

Desarrollo de sitio comercial para empresa dedicada a traducciones de documentos, el sitio incluye un formulario para cotizaciones.

Tecnologias usadas: WP (Elementor) y Vue.js

Foods Restaurant

Esta es una sencilla aplicacion donde se pone aprueba las multiples cualidades que ofrece tailwind.

Tecnologias usadas: HTML + Tailwind
